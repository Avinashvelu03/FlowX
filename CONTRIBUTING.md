# Contributing to flowx-control

Thank you! 🎉

## Getting Started

1. Fork & clone
2. `npm install` or `bun install`
3. `git checkout -b feat/your-feature`

## Development

- Tests: `npm test`
- Build: `npm run build`

## Guidelines

- 100% test coverage for new primitives
- Zero dependencies
- All patterns must be tree-shakable
